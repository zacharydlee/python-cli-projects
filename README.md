# Python CLI Projects
Collection of command-line applications built while learning core programming concepts. 

## 🚀 Projects

### ☕ Coffee Bot
A console-based ordering system that simulates a café experience.

**Features:**
- Menu selection with multiple drink options
- Size customization
- Looping order system for multiple items
- Input validation and user prompts

### 🏰 Dungeon Adventure
A text-based adventure game where the player explores a dungeon, manages resources, and makes decisions. 

**Features:**
- Exploration system with random events
- HP and gold management
- Shop and rest mechanics
- Game loop with win/lose conditions

## 🧠 Concepts Practiced
- Control flow (loops, conditionals)
- Functions and modular design
- User input handling
- Basic state management
- Randomized events

## ▶ How to Run
Run either program using Python:

```bash
python coffee_bot.py
```
or
```bash
python dungeon_adventure.py
```
